# G-Jobs — Connecting Job Seekers & Employers in Algeria 🇩🇿

![logo](https://raw.githubusercontent.com/zineb-hub/G-JOB/refs/heads/main/WITH-BG/3.png)

**G-Jobs** is a job-matching web platform designed to connect Algerian **job seekers and employers** — from full-time careers to side jobs and everyday services (babysitting, housekeeping, and more). It streamlines the search with profiles, smart filtering, messaging, and job tracking.

## Features

- **Dual user roles** — job seekers and job providers
- **Job posting & search** with filtering by type, location, and category
- **Profiles** for candidates and employers, with profile editing
- **Messaging** between seekers and providers
- **Job tracking** to follow applications and offers

## Tech Stack

- PHP (server-side)
- HTML, CSS, JavaScript (front end)
- MySQL database

## Getting Started

```bash
git clone https://github.com/ZinebMEFTAH/G-JOB.git
```

1. Deploy the project under a PHP server (e.g. XAMPP / Apache).
2. Import the database schema into MySQL.
3. Configure the database connection.
4. Open the site in your browser.
